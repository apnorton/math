static int math.NumberTheory.inverseFibonacci ( long  f)
static

This is an inverse for Binet's formula.

Formula from Math.SE:

\[ n=\left[ \log_\phi \sqrt{5}(F_n-\frac{1}{2}) \right],\quad n \ge 3\]

(where [x] is the rounding function.) I've manually corrected the case where \(0\le n < 3\), and negative inputs return -1.

It is safe to return an integer, because all long values are less than fib(93).

Parameters
fthe input number
Returns
the index of the largest Fibonacci number less than or equal to f.
